A simple paint app made with HTML5, CSS and JavaScript. 
It is a beginner friendly project for those who want to start with a project that is very interactive. 
This project exposes the programmer to the amazing world of Canvas API of HTML5 and how to manipulate it. 
This project also explores another unique way of adding an active class to an element. This is achieved by using an array and a ternary operator.
This is the link to the website https://shemmy01-web.github.io/PAINT-APP/
